1. Sets

Set theory is the mathematical study of collections of objects. A set is a well-defined collection of distinct objects, considered as an object in its own right. Sets are usually denoted by capital letters and their elements listed within curly braces { }. The language of sets allows us to discuss relationships between groups of items in a precise way.

Concept of a Set: A set is simply a collection of distinct objects, called elements or members of the set. If an element x is in set A, we write x ∈ A; if not, x ∉ A. Sets can be described by listing their elements or by a defining property. Examples:

A = {2, 4, 6, 8} is the set of first four positive even numbers.

B = {a, e, i, o, u} is the set of vowels in the English alphabet.

C = {x | x < 0 and x ∈ ℤ} describes the set of all negative integers (this is set-builder notation, reading as ‚Äúthe set of all x such that x is less than 0 and x is an integer‚Äù).

A set can also be infinite, e.g., ℕ = {1, 2, 3, ...} the set of natural numbers.

Set Language and Notation:

We use curly braces {} to list elements, and an ellipsis ‚Äú‚Ä¶‚Äù for continuing patterns (e.g., {1,2,3,...,100}).

The symbol ∈ means ‚Äúis an element of,‚Äù and ∉ means ‚Äúis not an element of.‚Äù

ℕ, ℤ, ℚ, ℝ often denote standard sets (natural numbers, integers, rationals, reals).

Two sets are equal (A = B) if they have exactly the same elements. Order and repetition in listing elements do not matter: {1,2,3} = {3,2,1} and {1,2,2,3} is the same as {1,2,3}.

Notation examples: If A = {1,2,3}, we can say 1 ∈ A, 4 ∉ A. We might also describe A = {x | 1 ≤ x ≤ 3, x ∈ ℕ}.

Empty Set: The empty set (or null set) is the set with no elements. It is denoted by {} or the symbol ∅ (phi). By definition, ∅ is a subset of every set (since there are no elements in ∅ that could violate subset conditions). Examples of empty sets:

The set of all humans with wings is ∅ (no such human exists).

{x | x^2 = 2 and x is rational} = ∅, because √2 is not rational.

Finite and Infinite Sets:

A set is finite if it has a limited number of elements. One can count the elements and eventually finish counting. (Example: {10, 20, 30} is finite with 3 elements.)

A set is infinite if it goes on without end. (Example: ℕ, the set of natural numbers, is infinite.) Infinite sets can be countably infinite (like ℕ or ℤ, where you can list elements in sequence) or uncountable (like ℝ, the real numbers, which cannot be listed in sequence without omission).

The cardinality of a set is the number of elements in it (for finite sets). We use notation |A| to denote the number of elements in A. If A = {2,4,6,8}, then |A| = 4. For infinite sets, |A| can be infinite (and in advanced contexts, different sizes of infinity can be compared).

Subsets and Equality of Sets:

A set A is a subset of set B (written A ⊆ B) if every element of A is also an element of B. If A is subset of B but A is not equal to B, sometimes we say A is a proper subset of B (A ⊂ B).

If A ⊆ B and B ⊆ A, that means A and B have exactly the same elements, hence A = B (equality of sets).

Examples:

{1,2} ⊆ {1,2,3} (and it‚Äôs a proper subset since 3 is in the larger set but not the smaller).

∅ ⊆ A for any set A (empty set is subset of all sets).

A ⊆ A (any set is a subset of itself, though that‚Äôs usually called an improper subset if we want to distinguish).

Cardinal Number of a Set: As mentioned, the cardinal number (cardinality) is the count of distinct elements in the set.

If |X| = n (a finite number), we can list those elements. If |X| = 0, X is empty.

Cardinality is used in probability and combinatorics often, to count outcomes. For infinite sets, cardinalities are denoted by concepts like ‚Ñµ‚ÇÄ (aleph-null) for countable infinity, etc., but at school level we focus on finite sets. Example: For A = {a, b, c}, |A| = 3. If B = {2,4,6,8}, |B| = 4. If C = {x | x is an integer, 1 ≤ x ≤ 100}, |C| = 100 (since it includes all integers 1 through 100).

Set Operations: Just as arithmetic has operations on numbers, there are operations to combine or relate sets:

Union (A ∪ B): The set of all elements that are in A or in B or in both. Essentially it combines the two sets (duplicates merged since sets don‚Äôt repeat elements). Example: If A = {1,2,3} and B = {3,4,5}, then A ∪ B = {1,2,3,4,5}.

Intersection (A ∩ B): The set of all elements that are common to both A and B (in A and in B). Example: With A and B as above, A ∩ B = {3} (only the element 3 is in both). If two sets have no element in common, their intersection is the empty set (they are called disjoint sets).

Difference (A ‚Äì B): The set of elements that are in A but not in B. Example: A ‚Äì B = {1,2} (elements in A excluding any that are also in B). B ‚Äì A = {4,5}.

Complement: Usually defined when talking about a universal set U (the set of all elements under consideration in a context). The complement of A (denoted A‚Ä≤ or A^c) is U ‚Äì A, i.e., all elements in the universe that are not in A. Example: If U = {1,2,3,4,5} and A = {1,2,3}, then A‚Ä≤ = {4,5}.

Cartesian Product (A √ó B): The set of all ordered pairs (a, b) where a ∈ A and b ∈ B. Example: If A = {1,2} and B = {x,y}, then A √ó B = {(1,x), (1,y), (2,x), (2,y)}. (This concept is a foundation for coordinate geometry and relations, though not explicitly in the list, it‚Äôs often taught alongside sets.)

Representation of Sets:

Roster/Tabular Form: Listing out all elements explicitly, when feasible. E.g., E = {2,4,6,8,10}.

Set-Builder Form: Describing the property of elements. E.g., E = {x | x is an even number between 1 and 11}.

Sometimes, we use Venn diagrams (see below) to represent sets visually, especially when dealing with operations or relationships among sets.

For large or infinite sets, roster form is impractical, so set-builder is useful (e.g., ℤ = {x | x is an integer}). Also, an ellipsis ‚Äú‚Ä¶‚Äù is used in roster form to indicate continuation.

Venn Diagrams and Their Properties: A Venn diagram is a visual way to represent sets and their relationships using shapes (usually circles or ovals) within a rectangle that often represents the universal set U. Overlapping regions indicate intersections, non-overlapping parts indicate differences, etc.

For two sets, a Venn diagram has two overlapping circles. The overlap is A ∩ B; the total area covered by both circles is A ∪ B. One can visually see which elements (or portions of the universal set) fall into which combination of membership.

For three sets, we use three overlapping circles, which can show all possible intersections (pairs or all three) etc.

Properties (laws) that can be illustrated via Venn diagrams include:

Commutative laws: A ∪ B = B ∪ A, and A ∩ B = B ∩ A (order doesn‚Äôt matter for union/intersection).

Associative laws: (A ∪ B) ∪ C = A ∪ (B ∪ C); (A ∩ B) ∩ C = A ∩ (B ∩ C) (grouping doesn‚Äôt matter).

Distributive laws: A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C); A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C).

De Morgan‚Äôs laws: The complement of a union is the intersection of complements, and vice versa: (A ∪ B)‚Ä≤ = A‚Ä≤ ∩ B‚Ä≤; (A ∩ B)‚Ä≤ = A‚Ä≤ ∪ B‚Ä≤.

Venn diagrams help in solving problems involving sets, like finding the number of elements in unions or intersections given some data (including survey problems with overlaps). They provide an intuitive understanding of set relations.

2. Algebra

Algebra is the branch of mathematics that uses symbols (usually letters) to represent numbers in expressions and equations. This allows us to generalize arithmetic and solve problems with unknown values. Algebra provides a powerful language to describe patterns and relationships. Key topics include:

Introduction to Algebra (Use of Symbols): In algebra, letters (like x, y, n) are used to stand for numbers. These letters are called variables because their values can vary or be unknown in general. For example, the equation x+5=12 asks "what number x would make this true?" (which we solve to find x=7). By using variables, we can state general formulas or relationships (e.g., the area of a circle A=πr2 works for any radius r).

Using variables, we can translate word problems into equations. For instance, "a number increased by 3 is 10" becomes x+3=10. Algebra is essentially a unifying thread in mathematics that helps solve equations, describe sequences, and define functions.

Algebraic Expressions: An algebraic expression is a combination of numbers, variables, and operations (such as +, -, ×, ÷, powers) but without an equality sign. Examples include: 2x+5, 3a2b-7, x2+3y, etc.

Terms: The parts of an expression separated by + or - signs. In 2x+5, the terms are 2x and 5. In 3a2b-7, the terms are 3a2b and -7.

Coefficients: The numerical factor of a term that contains a variable. In 2x, 2 is the coefficient of x. If no number is written, the coefficient is 1 (e.g., in x2, coefficient is 1).

Like and Unlike Terms: Terms are like terms if they have the exact same variable part (same variables raised to the same powers). They can be combined by addition/subtraction. E.g., 5x and 3x are like terms (they combine to 8x), but 5x and 5x2 are unlike (different powers).

Polynomials: A specific type of algebraic expression where all exponents are whole numbers and no variables in denominators. For example, x2+3x-2 is a polynomial in x. (Polynomials are covered more below.)

Exponents and Powers (in Algebraic context): Variables can also have exponents. The rules for exponents (mentioned in Number System above) apply in algebra too. For instance, x2×x3=x5.

Often in algebra, we use exponents to represent very large or small numbers in compact form (scientific notation) or to represent products of repeated factors algebraically.

We can only directly combine exponential terms by multiplication/division if they have the same base (or sometimes by same exponent in factorization contexts).

Also, note am×an=am+n and aman=am-n still hold.

With variables, an expression like 2x3 means 2×x×x×x. If we have something like x2×y3, it's x×x×y×y×y. If bases differ (e.g., x2×y2), you can combine them as (xy)2.

Factorization: Factorization (or factoring) is the process of expressing an algebraic expression as a product of its factors. It's essentially the reverse of expansion (multiplying out). Key methods of factorization include:

Common Factor Extraction: Identify a common factor in all terms and factor it out. Example: 6x2+9x=3x(2x+3) (we factored out 3x).

Grouping: For four-term expressions, sometimes grouping terms into pairs helps. Example: ax+ay+bx+by=a(x+y)+b(x+y)=(a+b)(x+y)

Using Identities (Special products): Recognize patterns such as difference of squares or perfect square trinomials (discussed below) to factor accordingly. Example: a2-b2=(a-b)(a+b) is an identity used to factor a difference of squares. If you had x2-9, you can factor it as (x-3)(x+3) because 9=32.

Quadratic Trinomials: For expressions like ax2+bx+c, find two numbers that multiply to ac and add to b (when a=1, it's simpler: multiply to c and add to b). Then split the middle term and factor by grouping. Example: To factor x2+5x+6, find two numbers multiplying to 6 and adding to 5 (that's 2 and 3). Higher-degree polynomials: Sometimes use synthetic division or long division to factor given one known root, using the Factor Theorem (if f(r)=0 then (x-r) is a factor). For example, if polynomial P(x)=x3-6x2+11x-6 has root x=2, then (x-2) is a factor, and you can divide P(x) by (x-2) to factor it. (This might be more advanced than required, but worth noting the concept.)

Special Products and Expansions: Certain algebraic expressions multiply out to well-known patterns, and remembering these can speed up both expansion and factorization:

Square of a Sum: (a+b)2=a2+2ab+b2

Square of a Difference: (a-b)2=a2-2ab+b2

Product of a Sum and Difference: (a+b)(a-b)=a2-b2 (Difference of two squares.)

Cube of a Sum: (a+b)3=a3+3a2b+3ab2+b3

Cube of a Difference: (a-b)3=a3-3a2b+3ab2-b3

Sum of Cubes: a3+b3=(a+b)(a2-ab+b2)

Difference of Cubes: a3-b3=(a-b)(a2+ab+b2)

General expansion: (x+a)(x+b)=x2+(a+b)x+ab (This is useful for mental expansion or factoring quadratics.) These identities are "special" because they occur often; recognizing them makes it easier to expand expressions quickly or factor expressions that match these forms. For example, x2-16 can be spotted as a difference of squares = (x-4)(x+4). Or if you need to expand (x-5)2, you can apply the second identity directly.

Linear Equations and their Graphs: A linear equation is an equation of degree 1 (the highest power of the variable is 1). In one variable (say x), a linear equation looks like ax+b=0 and has a single solution x=-ba. In two variables (x and y), a typical form is ax+by+c=0 (or y=mx+c), which represents a straight line when graphed on the coordinate plane. Key points:

Solution of a linear equation in one variable: There is exactly one solution (if a≠0). E.g., solve 2x + 7 = 15 ‚Üí 2x = 8 ‚Üí x = 4.

Solution of a linear equation in two variables: Infinitely many solutions (each solution is an (x,y) pair that lies on the line). For example, x+y=5 has solutions like (0,5), (1,4), (2,3), etc., forming a line.

Graphing a line: You can find two or more solutions (points) and plot them, then draw a straight line through. For example, to graph 2x+3y=6, find intercepts: set x=0 ‚Üí 3y=6 ‚Üí y=2 (point (0,2)); set y=0 ‚Üí 2x=6 ‚Üí x=3 (point (3,0)). Plot (0,2) and (3,0) and draw the line through them.

Intercept form: Lines often intercept axes; x-intercept is where y=0, y-intercept where x=0. The line equation can be expressed as xa+yb=1 if a and b are intercepts on x and y axes respectively.

Slope-intercept form: y=mx+c, where m is the slope of the line and c is the y-intercept (point where line crosses y-axis). e.g. y=2x+3 has slope 2, y-intercept 3. (Slope is covered more below.)

Polynomials: A polynomial is an expression consisting of variables and coefficients combined using only addition, subtraction, and multiplication (no division by a variable, and exponents are non-negative integers). Examples: 3x2-2x+1, x3+2x2-x+5, or even 7 (a constant is a polynomial of degree 0).

Degree: The degree of a polynomial is the highest exponent of the variable. E.g., 3x4-2x2+x-7 has degree 4. If multiple variables, degree is the sum of exponents in the highest term (e.g., 3x2y3 has degree 5 in two variables).

Classification by terms: A polynomial can be monomial (1 term, e.g. 5x3), binomial (2 terms, e.g. x2-4), trinomial (3 terms, e.g. 2x2+3x+1), etc.

Operations: Polynomials can be added, subtracted, and multiplied to get another polynomial. They can be divided as well, though not always a polynomial quotient (sometimes a remainder is left, similar to integer division).

Remainder & Factor Theorem: If a polynomial P(x) is divided by (x - a), the remainder is P(a). If P(a)=0, then (x - a) is a factor of P(x). This is a useful concept for finding roots or factoring higher-degree polynomials.

Zeros/Roots: The solutions to P(x)=0 are called roots of the polynomial. A polynomial of degree n has at most n real roots. For quadratic (degree 2) or higher, finding roots may require factorization or formulas.

Factoring: If the quadratic factors nicely: e.g., x2-5x+6 factors to (x-2)(x-3)=0, giving roots x=2 or x=3.

Concept of Progressions (Sequences): A progression is essentially a sequence of numbers that follow a specific pattern. Each number in the sequence is called a term. Progressions allow us to describe and analyze ordered sets of numbers where there is a definite rule connecting successive terms. Two common types are Arithmetic and Geometric progressions:

Arithmetic Progression (AP): An AP is a sequence where each term (after the first) is obtained by adding a fixed number called the common difference (d) to the previous term. Example: 5, 8, 11, 14, ‚Ä¶ is an AP with first term a‚ÇÅ = 5 and common difference d = 3.

nth term of an AP: an=a1+(n-1)d. This formula gives the value of the term at position n. For the example, the 10th term would be a10=5+(10-1)×3=32.

Example: Sum of first 5 terms of 5,8,11,...: first term 5, fifth term a5=5+(5-1)×3=17. Average of first and fifth = (5+17)/2 = 11, times 5 terms gives S_5 = 55. Listing them: 5+8+11+14+17 = 55, which matches.

APs are used in situations with constant change per step, like linear growth or uniform spacing (e.g., house numbers on a street, days of week repeating every 7 days which is an AP modulo 7).

nth term of a GP: an=a1rn-1. For the example, the 5th term would be a5=3×25-1=48.

GPs model exponential growth or decay situations (like population growth at a fixed percentage, financial interest compounding if deposits or ratios considered, radioactive decay, etc.).

3. Number System

The Number System covers different categories of numbers and their properties, forming the foundation of arithmetic and algebra. Important concepts include:

Prime and Composite Numbers:

Prime numbers are natural numbers greater than 1 that have exactly two factors: 1 and themselves. (Examples: 2, 3, 5, 7 are prime.)

Composite numbers have more than two factors (they can be divided evenly by numbers other than 1 and themselves). (Examples: 4 = 2√ó2, 6 = 2√ó3, 8 = 2√ó4 are composite.)

The number 1 is neither prime nor composite (it has only one factor, itself).

Fundamental Theorem of Arithmetic: Every composite number can be expressed as a product of prime numbers, and this factorization is unique (except for the order of factors). Example: 12 = 22×3, 60 = 22×3×5

Sieve of Eratosthenes: An ancient algorithm for finding all prime numbers up to a specified integer. It works by iteratively marking as composite (i.e., not prime) the multiples of each prime, starting with the first prime number, 2. This method efficiently eliminates non-prime numbers, leaving only primes unmarked.

Tests of Divisibility: These are quick rules to determine if a number is divisible by another without performing full division:

Divisible by 2: if the last digit is even (0, 2, 4, 6, 8).

Divisible by 3: if the sum of its digits is divisible by 3. (Example: 123 ‚Üí 1+2+3=6, and 6 is divisible by 3, so 123 is divisible by 3.)

Divisible by 4: if the last two digits form a number divisible by 4. (Example: 712 ‚Üí last two digits 12, and 12 is divisible by 4.)

Divisible by 5: if the last digit is 0 or 5.

Divisible by 6: if it is divisible by both 2 and 3 (combines those rules).

Divisible by 7: Double the last digit and subtract it from the rest of the number. If the result is divisible by 7 (including 0), then the original number is divisible by 7. (Example: 357 ‚Üí 35 - 2√ó7 = 35 - 14 = 21, which is divisible by 7, so 357 is divisible by 7.)

Divisible by 8: if the last three digits form a number divisible by 8.

Divisible by 9: if the sum of its digits is divisible by 9.

Divisible by 10: if the last digit is 0.

Divisible by 11: if the difference between the sum of digits in odd places and even places is 0 or a multiple of 11. (Example: 2728 ‚Üí (2+2) ‚Äì (7+8) = 4 ‚Äì 15 = -11, which is divisible by 11, so 2728 is divisible by 11.)

Divisible by 12: if it is divisible by both 3 and 4.

Whole Numbers and Integers:

Whole numbers are the non-negative integers: 0, 1, 2, 3, ‚Ä¶. (This set includes zero and all positive counting numbers.)

Integers include all whole numbers and their negatives: ‚Ä¶, -3, -2, -1, 0, 1, 2, 3, ‚Ä¶ . Integers extend indefinitely in both positive and negative directions on the number line.

Fractions and Decimal Fractions:

A fraction represents a part of a whole and is written in the form p/q (where q≠0). For example, ¬æ means 3 parts out of 4 equal parts of something. Fractions can be proper (numerator < denominator), improper (numerator ≥ denominator), or mixed (an integer with a fractional part, e.g. 2‚Öì).

Decimal fractions are another way to represent fractions, using place value and a decimal point. For example, 3/4 can be written as 0.75 in decimal form. Terminating decimals have a finite number of decimal places (0.5, 0.75), while repeating (recurring) decimals have one or more repeating digits or patterns (e.g. 0.333‚Ä¶ = 1/3, 0.272727‚Ä¶ = 3/11).

Least Common Multiple (LCM) and Greatest Common Divisor (GCD):

The LCM of two or more integers is the smallest positive number that is a multiple of each of the integers. Example: LCM(6, 8) = 24 (since 24 is the smallest number divisible by both 6 and 8). LCM is useful for adding or subtracting fractions with different denominators, or finding common periods in repeating events.

The GCD (also called HCF - Highest Common Factor) of two or more integers is the largest positive number that divides each of the integers without a remainder. Example: GCD(18, 24) = 6 (since 6 is the greatest number that divides both 18 and 24 exactly). GCD is useful for simplifying fractions to lowest terms.

Relation: For any two positive integers a and b, a×b=LCM(a,b)×GCD(a,b).

Prime Factorization Method: To find LCM and GCD using prime factorization:

  • For GCD: Take the product of the lowest powers of common prime factors.
  • For LCM: Take the product of the highest powers of all prime factors.

Example: For 12 = 22×3 and 18 = 2×32:
GCD(12, 18) = 21×31=6
LCM(12, 18) = 22×32=36

Rational and Irrational Numbers:

A rational number is a number that can be expressed as a fraction p/q where p and q are integers and q≠0. Rational numbers include integers (e.g. 5 = 5/1), terminating decimals (0.25 = 1/4), and repeating decimals (0.333‚Ä¶ = 1/3).

An irrational number cannot be expressed as a simple fraction; its decimal form is non-terminating and non-repeating. Classic examples include 2 (approximately 1.41421356‚Ä¶, which cannot be expressed exactly as a fraction) and π (3.14159‚Ä¶, the ratio of a circle's circumference to its diameter).

Real numbers consist of all rational and irrational numbers together ‚Äì essentially any number that can be found on the continuous number line.

Operations with Irrational Numbers:

  • The sum or difference of a rational and an irrational number is irrational.
  • The product or quotient of a non-zero rational number and an irrational number is irrational.
  • The sum, difference, product, or quotient of two irrational numbers may be rational or irrational.

Examples: 2+3 is irrational; 2×8=16=4 is rational.

Properties of Numbers (Closure, Commutativity, Associativity, etc.): These refer to fundamental properties of operations on number sets:

Closure: A set of numbers is closed under an operation if performing that operation on members of the set always yields a member of the same set. (Example: Integers are closed under addition and multiplication ‚Äì adding or multiplying any two integers gives an integer. However, integers are not closed under division, since 5 √∑ 2 = 2.5, which is not an integer.)

Commutative Property: Changing the order of operands does not change the result. Addition and multiplication are commutative for real numbers: a + b = b + a, a √ó b = b √ó a. (Subtraction and division are not commutative: e.g., 5 ‚Äì 2 ≠ 2 ‚Äì 5.)

Associative Property: Changing the grouping of operands (without changing the order) does not change the result. For addition and multiplication: (a + b) + c = a + (b + c), (a √ó b) √ó c = a √ó (b √ó c). (Again, subtraction/division are not associative.)

Distributive Property: Relates multiplication and addition: a √ó (b + c) = a√ób + a√óc. This property allows expansion of products over sums and is fundamental in algebra.

Identity Elements: For addition, 0 is the additive identity (adding 0 leaves a number unchanged: a + 0 = a). For multiplication, 1 is the multiplicative identity (a √ó 1 = a).

Inverse Elements: For addition, the inverse of a number is its negative (a + (‚àía) = 0). For multiplication, the inverse (reciprocal) of a nonzero number a is 1/a (since a √ó 1/a = 1). These allow solving equations by ‚Äúundoing‚Äù operations.

Real Numbers and the Number Line: All rational and irrational numbers together form the set of real numbers. Real numbers can be represented as points on the number line. Every real number corresponds to a unique point on the line, and conversely, every point on the line represents a unique real number. This continuum property of real numbers underlies the concept of distance and measurement in geometry and algebra.

Laws of Exponents (Indices): Exponents (or powers) denote repeated multiplication. The laws of exponents make calculation easier and algebraic manipulation possible: For any non-zero base a, and any integers m and n:

Product of powers: am×an=am+n. (Example: 23×24=27=128)

Quotient of powers: aman=am-n. (Example: 5552=53)

Power of a power: (am)n=am×n. (Example: (32)4=38)

Squares, Square Roots, Cubes, and Cube Roots:

Euclid‚Äôs Division Lemma: This is a formal statement of the long-division process for integers, and it is the basis of the Euclidean algorithm for finding GCD. It states that for any two integers a and b (with b > 0), there exist unique integers q and r such that: a = b ¬∑ q + r, where 0 ≤ r < b. Here, a is the dividend, b is the divisor, q is the quotient, and r is the remainder when a is divided by b. Example: If a = 20 and b = 6, we can write 20 = 6√ó3 + 2, where quotient q=3 and remainder r=2 (since 2 < 6). This lemma guarantees the existence of such q and r and that they are unique for given a and b.

Using Euclid‚Äôs division lemma repeatedly (Euclidean algorithm) allows us to find the GCD of two numbers. For instance, to find GCD(20, 6): 20 = 6√ó3 + 2, then 6 = 2√ó3 + 0; once remainder is 0, the last non-zero remainder (2) is the GCD.

Concept of Logarithms: A logarithm is essentially an exponent. The logarithm of a number y with respect to base b (written as log_b(y)) is the power to which b must be raised to produce y. In equation form: logb(y)=x if and only if bx=y Some key points and properties:

Laws of Logarithms:

  • logb(M×N)=logb(M)+logb(N) (the log of a product is the sum of logs)
  • logb(MN)=logb(M)-logb(N) (the log of a quotient is the difference of logs)
  • logb(Mp)=p×logb(M) (the log of a power is the exponent times the log of the base)

These laws mirror the laws of exponents and facilitate simplifying or solving exponential equations.

Change of Base Formula: logb(a)=logc(a)logc(b), which allows computing logs on calculators by changing to base 10 or base e.

Characteristic and Mantissa (for common logs): In log_10 form, the integer part of the logarithm is called the characteristic (indicates order of magnitude) and the fractional part is the mantissa (which depends on significant figures of the number). This is more relevant in using log tables.

Usage: Logarithms help solve equations where the unknown is in an exponent (e.g., find t if 2t=40, here t=log2(40)). They are also used to define scales like pH in chemistry or the Richter scale for earthquakes, where the logarithm maps large ranges of input into manageable numbers.

Solving Logarithmic Equations:

To solve equations involving logarithms:

  1. Isolate the logarithmic expression
  2. Apply the definition of logarithms to convert to exponential form
  3. Solve the resulting equation
  4. Check that solutions are valid (arguments of logarithms must be positive)

Example: Solve log2(x-3)=4

Simplify: 16=x-3
Solve: x=19
Check: x-3=16>0, so the solution is valid.

4. Arithmetic

Arithmetic deals with numerical calculations and the application of the basic operations (addition, subtraction, multiplication, division) to solve problems, often related to real-life scenarios. Key topics include:

Ratio and Proportion:

A ratio is a comparison of two quantities by division. It is expressed as ‚Äúa to b‚Äù or a:b (which equals a/b as a fraction). For example, if there are 2 apples and 3 oranges, the ratio of apples to oranges is 2:3. Ratios can be scaled up or down (2:3 is equivalent to 4:6, 6:9, etc.).

A proportion states that two ratios are equal. For instance, 2:3 = 6:9 is a proportion. In equation form, a/b = c/d. A useful property: in a true proportion, cross-multiplying gives ad = bc.

Problem-solving: If a/b = c/d and one of the four values is unknown, you can find it by cross-multiplication. Example: If a recipe mixes sugar and flour in ratio 2:5 and you have 10 cups of flour, how much sugar is needed? Set 2/5 = x/10, cross-multiply to get 5x = 20 ‚Üí x = 4 cups of sugar.

Simple Interest: Simple interest is interest calculated only on the principal (the initial amount of money) over time. It grows linearly.

Formula: I = (P √ó R √ó T) / 100, where: P = principal (initial amount), R = annual interest rate (in %), T = time period (in years).

Interest vs. Amount: The interest I is the extra money earned or paid. The total amount A after interest is added: A = P + I.

Example: If P = ‚Çπ1000, R = 5% per year, and T = 3 years, then I = (1000√ó5√ó3)/100 = ‚Çπ150. The amount A = 1000 + 150 = ‚Çπ1150.

Simple interest is used in situations like short-term loans, simple bank accounts, or bonds when compounding is not considered.

Compound Interest: Compound interest is interest calculated on the initial principal and also on the accumulated interest from previous periods. This means the money grows at an increasing rate (interest on interest).

Annual compounding formula: A = P √ó (1 + R/100)^T, where A is the amount after T years (with interest added). The compound interest earned would be A ‚Äì P.

If compounding is more frequent (like quarterly, monthly), the formula adjusts: for compounding n times a year at rate R, A = P √ó (1 + (R/(100n)))^(nT).

Example: P = ‚Çπ1000 at 10% annual rate, compounded yearly for 2 years: A = 1000√ó(1+0.10)^2 = 1000√ó1.1^2 = 1000√ó1.21 = ‚Çπ1210. The interest earned = ‚Çπ210. (Notice this is slightly more than the ‚Çπ200 that simple interest would give, due to interest-on-interest in the second year.)

Compound interest is used in savings accounts, investments, and any situations where interest is periodically added to the principal. It grows faster than simple interest for the same rate and time.

Time and Distance: This involves problems relating speed, distance, and time, which are connected by the formula Distance = Speed √ó Time.

If you know any two of these quantities, you can find the third: Speed = Distance/Time, Time = Distance/Speed.

Units must be consistent (e.g., if distance is in km and time in hours, speed will be in km/h). Conversion may be needed (e.g., km/h to m/s or vice versa: 1 m/s = 3.6 km/h).

Example: A car traveling at 60 km/h for 2.5 hours will cover Distance = 60 √ó 2.5 = 150 km.

Relative speed: If two objects move in the same direction, their relative speed is the difference of their speeds; if in opposite directions, it‚Äôs the sum. This concept is useful in problems about moving trains, overtaking, head-on approach, etc.

Average speed: If an object covers different distances with different speeds, average speed = total distance / total time (not simply the average of speeds). For example, going to a place at 30 km/h and returning at 60 km/h, the average speed is not 45 km/h; instead, use total distance and time to compute it.

Discount: Discount is a reduction on the marked price of an item, often given as a percentage. It‚Äôs commonly seen in shopping or commerce.

Marked Price (List Price): The original price of the item before discount.

Discount Rate: A percentage of the marked price by which the price is reduced.

Sale Price (Discounted Price): The price after subtracting the discount. Sale Price = Marked Price ‚Äì Discount. Also, if the discount rate is D%, then Discount = (D/100) √ó Marked Price.

Example: An article marked at ‚Çπ500 is on 10% discount. Discount = 10/100 √ó 500 = ‚Çπ50. So the sale price = 500 ‚Äì 50 = ‚Çπ450.

Sometimes successive discounts are given (e.g., 20% off, then an additional 10% off on the reduced price). These are not simply additive (20%+10%=30% total) but multiplicative in effect: the final price = original price √ó (0.80) √ó (0.90) = original √ó 0.72, effectively a 28% overall discount.

Tax: In everyday contexts, ‚Äútax‚Äù often refers to sales tax or value-added tax (VAT/GST) applied on purchases, or income tax on earnings. Here we focus on sales tax (or similar) in price calculations:

Sales Tax/Value Added Tax: A percentage added to the selling price of goods/services. If an item‚Äôs base price is P and tax rate is T%, the tax amount = (T/100)√óP, and Total Price = P + Tax.

Example: A gadget costs ‚Çπ1000 before tax, and the sales tax rate is 5%. Tax = 0.05√ó1000 = ‚Çπ50, so total price = ‚Çπ1050.

Sometimes prices are quoted ‚Äúinclusive of tax,‚Äù meaning tax is already included. To find the base price from an inclusive price, one can reverse-calculate: Base Price = Total Price / (1 + T/100).

Income Tax: calculated on earnings as per certain slabs or rates, but that‚Äôs outside basic arithmetic scope.

Time and Work: This topic deals with how long it takes for one or more people (or machines) to complete a task, and how they work together. The key is understanding work rate.

Define ‚Äúone unit of work‚Äù as the whole job. If a person can complete the job in D days, then the person‚Äôs work rate is 1/D of the job per day.

If two people have rates R1 and R2 (jobs per day), working together their combined rate is R1 + R2 jobs per day (assuming they don‚Äôt interfere with each other). The time taken together for one job = 1/(R1+R2) days.

Example: If A can finish a job in 5 days and B can finish the same job in 10 days, then A‚Äôs rate = 1/5 job per day, B‚Äôs rate = 1/10 per day. Together rate = 1/5 + 1/10 = 3/10 jobs per day. So together they finish the job in 10/3 ‚âà 3.33 days.

Work equivalence: Sometimes work problems involve different units or outputs (like ‚Äúpipes filling a tank‚Äù, ‚Äúmachines producing items‚Äù). The concept is similar: find individual rates (tank per hour, items per hour, etc.), then combine.

Inverse relationship: More people/machines working together means less time needed, roughly inversely proportional if all have equal efficiency (e.g., if one person takes 6 days, two similar people take ~3 days).

Profit and Loss: These concepts apply to buying and selling of goods, comparing cost and selling price to determine gain or loss.

Cost Price (CP): The price at which an item is purchased (cost to the seller).

Selling Price (SP): The price at which the item is sold to a customer.

Profit (or Gain): If SP > CP, the seller makes a profit = SP ‚Äì CP.

Loss: If SP < CP, the seller incurs a loss = CP ‚Äì SP.

Profit/Loss Percentage: This is usually calculated with respect to cost price:

Profit% = (Profit / CP) √ó 100

Loss% = (Loss / CP) √ó 100

Example: A shopkeeper buys an item for ‚Çπ80 (CP) and sells it for ‚Çπ100 (SP). Here Profit = ‚Çπ20, and Profit% = (20/80)√ó100 = 25%. If instead the selling price was ‚Çπ70, Loss = ‚Çπ10, Loss% = (10/80)√ó100 = 12.5%.

Mark-up and Discount interplay: A seller might mark an item at a higher price than CP and then give a discount. Understanding profit and loss in such cases requires careful step-by-step calculation (first apply markup to find SP before discount, then apply discount, etc.).

Break-even: SP = CP results in neither profit nor loss (0% gain).

5. Geometry

Geometry is the study of shapes, sizes, and properties of space. It spans from flat shapes (plane geometry) to solid shapes (solid geometry). Early geometry was developed to solve practical problems in measurement, construction, and astronomy. This section covers fundamental geometry concepts, some historical context, and coordinate geometry which bridges algebra and geometry.

History of Geometry: Geometry has ancient origins in various civilizations. Practical geometry (for land surveying, construction) existed in Egypt and Mesopotamia. The Greeks systematized geometry: Euclid (around 300 BCE) wrote The Elements, a collection of geometric knowledge organized axiomatically, which became the basis for Euclidean geometry taught in schools.

Euclid‚Äôs approach: He started with basic definitions (point, line, etc.), postulates (like ‚Äúthrough any two points there is exactly one straight line‚Äù), and common notions, then systematically proved a sequence of propositions. This logical development set the standard for deductive reasoning in math.

Indian contributions: Ancient India also contributed significantly. The ≈öulba S≈´tras (dated ~800-500 BCE) contained geometric constructions and the earliest recorded statement of the Pythagorean theorem (as a rule for altar construction). Indian mathematicians like Aryabhata and Brahmagupta (5th-7th century CE) worked on astronomy and geometry; Brahmagupta derived formulas for cyclic quadrilaterals (Brahmagupta‚Äôs formula gives area = √((s-a)(s-b)(s-c)(s-d)) for a quadrilateral inscribed in a circle, where s is semiperimeter). Indian mathematicians also developed concepts of trigonometry which link to geometry of circles.

Other contributions: The Arabs preserved and expanded geometric knowledge during the medieval period (introducing algebraic techniques into geometry), and later, the development of coordinate geometry by Ren√© Descartes (1637) merged algebra and geometry, revolutionizing the field. Geometry continued to evolve with non-Euclidean geometries in the 19th century (though that‚Äôs beyond school scope).

Euclid‚Äôs Geometry (Euclidean Geometry): Euclidean geometry refers to the geometry of the plane and space based on Euclid‚Äôs postulates. Key ideas include:

Points and Lines: A point indicates a position (no size). A line has length but no thickness and extends infinitely in both directions. A line segment has two endpoints and finite length. A ray starts at a point and extends infinitely in one direction.

Postulates (simplified):

A straight line segment can be drawn joining any two points.

A straight line can be extended indefinitely in a straight line.

A circle can be drawn with any center and radius.

All right angles are congruent.

(Parallel Postulate) Given a line and a point not on it, there is exactly one line through the point parallel to the given line.

Using these and other logical axioms, Euclid derived many propositions (theorems) about triangles, parallel lines, circles, etc. In Euclidean geometry: the sum of angles in a triangle is 180¬∞, the parallel postulate holds true, and concepts like congruence and similarity are defined via rigid motions or ratios.

Euclidean vs Non-Euclidean: In non-Euclidean geometries, the parallel postulate is altered; for instance, on a sphere, ‚Äúlines‚Äù are great circles and parallels as we know them don‚Äôt exist (sum of angles in a triangle > 180¬∞). But for school geometry, we stick to Euclidean geometry (flat plane, typical rules).

Lines and Angles: Fundamental building blocks in geometry are lines and angles formed when lines intersect or when we construct shapes:

Line relationships: Lines can be intersecting (cross at a point) or parallel (never meet, always same distance apart). If lines are perpendicular, they intersect at a right angle (90¬∞).

Angles: An angle is formed by two rays sharing a common endpoint (vertex). Measured in degrees: a full circle is 360¬∞. Types of angles:

Acute angle: less than 90¬∞

Right angle: exactly 90¬∞

Obtuse angle: between 90¬∞ and 180¬∞

Straight angle: 180¬∞ (a straight line)

Reflex angle: between 180¬∞ and 360¬∞

Pairs of angles:

Adjacent angles: share a vertex and a side but do not overlap.

Vertical (opposite) angles: formed when two lines intersect; they are opposite each other at the intersection. Vertical angles are always equal.

Complementary angles: two angles that sum to 90¬∞ (one is the complement of the other).

Supplementary angles: two angles that sum to 180¬∞ (one is the supplement of the other). Adjacent supplementary angles on a straight line are called a linear pair.

Alternate interior angles: when two parallel lines are cut by a transversal (a line crossing them), angles on opposite sides of the transversal and inside the parallel lines are alternate interior angles (they are equal if lines are parallel).

Corresponding angles: in the same scenario, an angle in the upper left position at one intersection corresponds to the angle in the upper left position at the other intersection; these are equal if lines are parallel.

These angle relationships are vital in proving lines parallel or analyzing geometric figures.

Similar Triangles: Triangles are similar if they have the same shape, but possibly different sizes. Formally, two triangles are similar if their corresponding angles are equal (and thus corresponding sides are proportional).

Criteria for similarity:

AAA (Angle-Angle-Angle): If two angles of one triangle equal two angles of another, the third automatically equals too, and the triangles are similar. (AA is usually enough to establish similarity, as the third angle will match.)

SAS (Side-Angle-Side in proportion): If an angle of one triangle equals an angle of another, and the sides including those angles are in the same ratio, the triangles are similar.

SSS (Side-Side-Side in proportion): If all three pairs of corresponding sides are in the same ratio, the triangles are similar.

Properties of similar triangles:

Corresponding sides are proportional: if triangles ABC ~ DEF, then AB/DE = BC/EF = CA/FD (and that common ratio is the scale factor of similarity).

Corresponding angles are equal: ‚à†A = ‚à†D, ‚à†B = ‚à†E, ‚à†C = ‚à†F in the example above.

If one triangle is scaled (enlarged or reduced) to form the other, lengths scale by factor k, areas scale by k2.

Applications: Using similarity is a powerful tool to find unknown lengths (e.g., shadow problems or using small similar triangles within big ones in geometric proofs). For instance, the concept behind map scales or models relies on similarity.

Pythagoras‚Äô Theorem: In any right-angled triangle (one angle = 90¬∞), the relationship between the side lengths is given by Pythagorean theorem: (Hypotenuse)^2 = (Leg)^2 + (Other Leg)^2. Typically phrased: c2=a2+b2, where c is the length of the hypotenuse (side opposite the right angle), and a and b are lengths of the other two sides.

Example: In a right triangle with legs 3 cm and 4 cm, the hypotenuse is 32+42=25=5 cm. (3-4-5 forms a Pythagorean triple.)

Converse of Pythagoras: If in a triangle a2+b2=c2 holds, then the triangle has a right angle opposite side of length c. This is useful to check if a triangle with given sides is right-angled.

The theorem is fundamental in geometry, allowing distance calculations. It‚Äôs also the basis for defining distance in coordinate geometry and has countless proofs (including one by Euclid). Ancient cultures like Babylon and India knew specific Pythagorean triples long before the general theorem was articulated.

Congruency of Triangles: Two figures are congruent if they are exactly the same shape and size. Congruent triangles have all three sides equal and all three angles equal (essentially one is a copy of the other, possibly rotated or flipped). To prove triangles congruent, certain minimum conditions can be met:

SSS (Side-Side-Side): If three sides of one triangle are respectively equal to three sides of another, the triangles are congruent.

SAS (Side-Angle-Side): If two sides and the included angle of one triangle are equal to the corresponding two sides and included angle of another, they are congruent.

ASA (Angle-Side-Angle): If two angles and the included side of one triangle equal the corresponding two angles and side of another, congruency holds.

AAS (Angle-Angle-Side): If two angles and a non-included side of one triangle equal the corresponding parts of another, the triangles are congruent. (This is a variation since if two angles are known, the third is also known, so it reduces to ASA effectively.)

RHS (Right angle-Hypotenuse-Side): In right triangles, if the hypotenuse and one leg of one triangle equal the hypotenuse and one leg of another right triangle, the two are congruent.

Congruent triangles have all corresponding sides equal and angles equal. Once triangles are known congruent, any corresponding measures (altitudes, medians, etc.) are also equal. This concept is used to prove many geometric properties (like showing two segments are equal by placing them as sides of congruent triangles).

Properties of Circles: A circle is the set of all points in a plane that are at a fixed distance (radius) from a fixed point (center). Important parts and properties include:

Radius (r): A segment from center to any point on the circle. All radii of a circle are equal.

Diameter (d): A chord that passes through the center. d=2r. It's the longest distance across the circle.

Circumference: The perimeter or length around the circle. Formula: C=2πr=πd

Arc: A connected part of the circle‚Äôs circumference. Measured in degrees (central angle) or length. A semicircle is an arc of 180¬∞.

Chord: A segment joining any two points on the circle. Diameter is a special chord (longest one).

Tangent: A line that touches the circle at exactly one point. A tangent is perpendicular to the radius at the point of contact.

Sector: A ‚Äúpie-slice‚Äù region bounded by two radii and the arc between them (like central angle area).

Segment (circular): Region between a chord and the arc above it. (A minor segment is the smaller area cut off by a chord, major segment the larger).

Angles in circles: Several theorems: e.g., angle in a semicircle is 90¬∞ (if you take any diameter and any third point on circle, the triangle formed is right-angled at that third point); angles at center vs circumference (central angle subtended by an arc is twice the inscribed angle on the circle subtended by the same arc). Opposite angles in a cyclic quadrilateral (all corners on the circle) sum to 180¬∞, etc.

Circle Area: A=πr2 For example, a circle of radius 7 cm has area π×72=49π.

Circular combinations: Annulus (ring) area = π(R2 - r2) if R is outer radius and r inner radius. These are just a few properties; circles have rich geometry, including symmetry (a circle has infinite lines of symmetry through its center and rotational symmetry of any angle).

Properties of Triangles: Triangles have many important properties and center points:

Angle Sum Property: Sum of interior angles of any triangle is 180¬∞. So if two angles are known, the third is 180¬∞ minus the sum of the two.

Exterior Angle Property: An exterior angle of a triangle (formed when one side is extended) equals the sum of the two opposite interior angles. (This is a consequence of the angle sum rule.)

Types of triangles by sides: Equilateral (3 equal sides, each angle 60¬∞), Isosceles (2 equal sides, base angles equal), Scalene (no equal sides).

Types by angles: Right (one 90¬∞ angle), Acute (all angles < 90¬∞), Obtuse (one angle > 90¬∞).

Perimeter: sum of side lengths.

Area: A=12×base×height. For example, if base = 10, height = 6, area = ¬Ω√ó10√ó6 = 30. Also Heron's formula for area if sides a, b, c: A=s(s-a)(s-b)(s-c), where s=a+b+c2 is semiperimeter.

Important centers:

Circumcenter (intersection of perpendicular bisectors of sides) ‚Äì center of the circumscribed circle through all vertices; in acute triangle it lies inside, right triangle at midpoint of hypotenuse, obtuse outside.

Orthocenter (intersection of altitudes) ‚Äì meets inside for acute, on the vertex of right angle for right triangles, outside for obtuse.

Inequality: In any triangle, sum of any two sides > third side. Also, the longest side is opposite the largest angle (and vice versa).

Congruence & similarity (already discussed) help derive many properties like segments or angles equal under certain conditions.

Properties of Quadrilaterals: A quadrilateral is a four-sided polygon. General properties and special types:

Angle sum: Interior angles of any quadrilateral sum to 360¬∞. (Because a quadrilateral can be divided into two triangles, each contributing 180¬∞).

Types of quadrilaterals:

Parallelogram: opposite sides are parallel (so two pairs of parallel sides). Opposite sides are equal in length, opposite angles equal. Diagonals bisect each other. Examples: rectangle, rhombus are special parallelograms.

Rectangle: a parallelogram with all angles 90¬∞. Opposite sides are equal; diagonals are equal in length.

Rhombus: a parallelogram with all sides equal (like a pushed over square). Opposite angles equal, diagonals perpendicular and bisect angles.

Square: a rectangle with all sides equal (also a rhombus with right angles). It has all properties of rectangle and rhombus (all sides equal, all angles 90¬∞, diagonals equal, bisect each other at 90¬∞, etc.).

Kite: two pairs of adjacent sides equal. One pair of opposite angles equal (the ones between unequal sides). Diagonals are perpendicular, one of them bisected by the other.

Area formulas:

Polygons (General Properties): A polygon is a closed figure with straight sides. A triangle (3 sides), quadrilateral (4 sides) are polygons, and so on. For an n-sided polygon (n-gon):

Parts of a Circle: (This was partly covered under circle properties, but here we list clearly parts and related terms)

Center: The given fixed point from which all points on the circle are equidistant.

Radius: A line segment from the center to any point on the circle; all radii are equal in a circle.

Diameter: A chord passing through the center; longest chord; equal to 2 radii.

Circumference: The distance around the circle (perimeter).

Arc: A part of the circumference. A minor arc is smaller than a semicircle; a major arc is larger.

Semicircle: An arc spanning half the circle (ends at endpoints of a diameter).

Chord: A segment joining two points on the circle. The diameter is a special chord.

Tangent: A line that touches the circle at exactly one point (point of tangency). A tangent is perpendicular to the radius at that point.

Secant: A line that cuts through the circle at two points (essentially extending a chord infinitely in both directions).

Central angle: An angle with its vertex at the center of the circle; its measure equals the measure of the intercepted arc.

Inscribed angle: An angle with its vertex on the circle (and rays along chords); its measure is half the measure of the intercepted arc.

Sector: The region bounded by two radii and the arc between them (like a ‚Äúpizza slice‚Äù of the circle).

Segment (of a circle): The region between a chord and the arc it subtends.

Concentric circles: Two or more circles with the same center but different radii.

Annulus: The ring-shaped region between two concentric circles (mentioned as ‚Äúring‚Äù in Mensuration section for area calculations).

Construction of Circle, Triangles, and Quadrilaterals: Geometric construction refers to drawing shapes accurately using a compass and straightedge (no measurement of lengths/angles with tools, just construction). Common constructions include:

Circle: Given a center point and a radius length, one can draw a circle with a compass set to that radius. Triangles: Various scenarios ‚Äì for example:

Construct a triangle given three side lengths (SSS). Use each length to draw arcs and find intersections for vertices.

Given two sides and the included angle (SAS), draw one side, then use a protractor or constructed angle, and mark off the second side from the angle, then join to form triangle.

Given two angles and a side (ASA/AAS), one can draw one side, then construct the angles at its ends, then mark off the second side length along one ray and complete the triangle.

Construct special triangles: e.g., an equilateral triangle (all sides equal) can be constructed by taking a segment and swinging arcs from its endpoints.

Circles and Concurrent Lines in Triangles:

Concurrency in Triangles: Several important sets of lines in a triangle meet at single points (are concurrent):

Medians: A median connects a vertex of a triangle to the midpoint of the opposite side. All three medians intersect at the centroid (as mentioned, centroid divides medians in 2:1 ratio from vertex).

Perpendicular Bisectors: These are lines that are perpendicular to each side at its midpoint. They all intersect at the circumcenter, which is equidistant from all vertices (center of the circumscribed circle around the triangle).

Angle Bisectors: Lines that split each angle into two equal angles. They meet at the incenter, which is equidistant from all sides (center of inscribed circle).

Altitudes: Perpendiculars dropped from each vertex to the opposite side (or its extension). They meet at the orthocenter.

These intersection points (centroid, circumcenter, incenter, orthocenter) are four notable ‚Äúcenters‚Äù of a triangle, each with significance. For example, the circumcenter allows one to draw a circle through all vertices, the incenter a circle tangent to all sides. In an equilateral triangle, all these four points coincide at the same spot.

Circles in Triangles:

Incircle: The circle inscribed in a triangle (touching all three sides) is centered at the incenter.

Circumcircle: The circle passing through all three vertices of a triangle is centered at the circumcenter.

Some triangle properties: all triangles have an incircle and circumcircle. A right triangle‚Äôs circumcenter lies at the midpoint of the hypotenuse (since the hypotenuse subtends a right angle at the circle‚Äôs circumference, meaning the hypotenuse is a diameter of the circumcircle).

Excircle: A circle externally tangent to one side of triangle and the extensions of the other two; there are three excircles (one opposite each vertex), centered at intersection of external angle bisectors.

Coordinate Geometry (Cartesian Geometry): Coordinate geometry introduces using ordered pairs of numbers to represent points on a plane. By doing so, algebraic methods can solve geometric problems. The Cartesian coordinate system consists of two perpendicular axes: the horizontal x-axis and vertical y-axis, intersecting at the origin (0,0). A point in the plane is given by coordinates (x, y) where x is the distance along the x-axis and y the distance along the y-axis.

Points on axes have either x=0 or y=0. Example: Plot (3, 2) (go right 3, up 2), (-4, 1) (left 4, up 1), ( -2, -3 ) (left 2, down 3). Reading coordinates: given a point on graph, see its projections on axes to determine (x,y).

Slope of a Line: The slope (m) of a line measures its steepness or inclination. It is defined as the ratio of the ‚Äúrise‚Äù (vertical change) to the ‚Äúrun‚Äù (horizontal change) between any two distinct points on the line.

A positive slope means the line rises as x increases (uphill left-to-right). A negative slope means the line falls as x increases (downhill left-to-right). Zero slope means the line is horizontal. Undefined slope (division by zero) corresponds to a vertical line (x constant).

In equation form y = mx + c, m is explicitly the slope.

Symmetry: Symmetry in geometry refers to a situation where a shape or figure is invariant under some transformation (reflection, rotation, etc.). Two major types in elementary geometry:

Line (Reflection) Symmetry: A figure has line symmetry if it can be reflected across a line (the line of symmetry) and appear unchanged. For instance, a circle has infinitely many lines of symmetry (any diameter line), a square has 4 (two axes through opposite sides and two through opposite vertices), a triangle may have some depending on type (equilateral has 3, isosceles has 1, scalene has none). In coordinate geometry, the y-axis is a line of symmetry for even functions (like y = x2), etc.

Rotational Symmetry: A figure has rotational symmetry if it can be rotated (by an angle less than 360¬∞) about a central point and coincide with itself. The number of rotations that map it to itself within 360¬∞ is the order of symmetry. For example, an equilateral triangle has rotational symmetry of order 3 (120¬∞ rotations), a square order 4 (90¬∞ rotations). A circle technically has infinite order rotational symmetry (any rotation works).

Point Symmetry (Origin symmetry): When rotated 180¬∞ about a point (often the origin in coordinate geometry), the figure looks the same (like an upside-down rotation flips it onto itself). For example, the graph of y = x3 is symmetric about the origin (point symmetry).

In coordinate terms, symmetry can be tested: e.g., symmetry about the y-axis means replacing (x,y) with (-x,y) yields the same equation/shape. Symmetry about the origin means (x,y) to (-x,-y) leaves shape invariant.

Applications: Recognizing symmetry can simplify problems ‚Äì e.g., in integration or geometry proofs, symmetrical properties can reduce complexity. In design and nature, symmetry is aesthetically and structurally significant.

6. Mensuration

Mensuration is the mathematics of measuring geometric figures, such as finding lengths, areas, and volumes. It provides formulas to quantify physical space that shapes occupy. Key formulas and concepts for 2D shapes (perimeter and area) and 3D solids (surface area and volume) include:

Perimeter of a Square and Rectangle:

Square: All four sides equal (side = a). Perimeter = 4a. For example, a square of side 5 cm has perimeter 20 cm.

Rectangle: Opposite sides equal (length = L, breadth = B). Perimeter = 2(L + B). E.g., rectangle with L=8, B=3 has perimeter 2*(8+3)=22 units.

Note: Perimeter is the total distance around the shape. Units are the same as the side lengths (e.g., cm, m).

Area of a Square and Rectangle:

Square: Area = side2 = a2. (Conceptually, it's how many unit squares can fit in the square.) Example: side 5 cm gives area 25 cm2.

Rectangle: Area = length √ó breadth = L √ó B. Example: 8 √ó 3 = 24 (square units).

If dimensions are in m, area in m2, etc. Simple but fundamental.

Area of a Triangle:

Area = ¬Ω √ó base √ó height. The base can be any side; the height (altitude) is the perpendicular distance from the opposite vertex to that base line.

Example: a triangle with base 10 cm and corresponding height 6 cm has area ¬Ω * 10 * 6 = 30 cm2.

Other formulas: If all sides a, b, c known, use Heron‚Äôs formula: (semiperimeter), then

For right triangles, a quick formula is ¬Ω * product of the two legs (since one leg can serve as base and the other as height, being perpendicular).

Area of a Circle:

Area = πr2, where r is radius. This formula comes from calculus or cutting circle arguments but is accepted as standard.

If diameter D given, since D = 2r, area = π(D2)2=π4D2

Example: radius 7 m ‚Üí area = π * 49 = 49π m2 (~153.94 m2 if π‚âà3.14).

A ring is the region between two concentric circles (one circle cut out of a larger circle). If the outer radius is R and inner radius is r (R > r), then Area of ring = Area_big_circle ‚Äì Area_small_circle = πR2 ‚Äì πr2 = π(R2 - r2).

This formula is just a direct application of circle area for difference of two radii.

Area of Quadrilaterals: There are various formulas depending on the type of quadrilateral:

Parallelogram: Area = base √ó height (very much like a slanted rectangle). The base is any side, and height is the perpendicular distance to the opposite side. Note: height is not necessarily the length of the other side unless it‚Äôs a rectangle. Example: parallelogram with base 12 cm and a height (altitude) of 5 cm has area 60 cm2 (even if the side length slanting is longer).

Trapezium (Trapezoid): One pair of parallel sides, say lengths a and b, and distance (height) h between them: Area = ¬Ω √ó (a + b) √ó h. (It‚Äôs the average of the parallel sides times the height.) Example: base1 = 10, base2 = 6, height = 4: area = 0.5(10+6)4 = 0.5164 = 32 units2.

Rhombus: All sides equal, but not generally 90¬∞ angles. With diagonals of lengths d1 and d2: Area = ¬Ω √ó d1 √ó d2. (Because diagonals bisect each other at right angles, effectively dividing the rhombus into 4 right triangles.) Example: d1=10, d2=8, area = 0.5108 = 40.

Rectangle/Square: already covered (special cases of parallelogram; square also special rhombus/rectangle).

Kite: like a rhombus formula essentially if you know diagonals (since kite also has perpendicular diagonals, one of which is bisected by the other typically), so area formula is also ¬Ω * d1 * d2 for a kite.

General Quadrilateral: If we know coordinates, can split into two triangles and use determinant method, or if we know one diagonal and the heights to that diagonal from the other two vertices, we can sum areas of two triangles. There isn‚Äôt a simple single formula for all general quadrilaterals with only side lengths (except cyclic quadrilateral or if other info given).

Polygons in general: If coordinates are known, can use shoelace formula for any polygon to find area.

Surface Area and Volume of a Cube:

Cube has 6 identical square faces (side length = a).

Total Surface Area (TSA): 6 * (area of one face) = 6a2.

Lateral (Side) Surface Area (LSA): Area of the 4 side faces (excluding top and bottom) = 4a2.

Volume: a3 (since length √ó width √ó height = aaa).

Example: side 3 cm ‚Üí TSA = 6*9 = 54 cm2; Volume = 27 cm3.

Surface Area and Volume of a Cuboid:

A cuboid (rectangular prism) has length L, breadth B, height H. It‚Äôs like a 3D rectangle.

Total Surface Area: sum of areas of 6 faces. Opposite faces are equal in area: two L√óB, two B√óH, two L√óH. So TSA = 2(LB + BH + LH).

Lateral Surface Area: only the four walls (excluding top and bottom): 2(L+B)H (because there are two pairs of opposite rectangular side faces each of dimension L√óH and B√óH). Another way: perimeter of base (rectangle) * height.

Volume: L √ó B √ó H (basically base area * height).

Example: L=5, B=4, H=3 ‚Üí TSA = 2(54 + 43 + 53) = 2(20 + 12 + 15) = 247 = 94; LSA = 2(5+4)3 = 293 = 54; Volume = 543 = 60.

Surface Area and Volume of a Cylinder:

A right circular cylinder has base radius r and height h. Think of a tin can shape.

Curved Surface Area (CSA or Lateral Surface): The area around the side, excluding top and bottom, which unrolls into a rectangle (width = circumference, height = h). CSA = circumference * height = 2πrh

Total Surface Area: includes top and bottom circles. So TSA = Lateral area + 2*(area of base) = 2πrh+2πr2=2πr(h+r) Sometimes factored as 2πr(h+r)

Volume: base area * height = πr2h (Because cross-section is a circle of area πr2, extruded height h.)

Example: r=5, h=10: CSA = 2π510 = 100π; base area = 25π, so TSA = 100π + 2*25π = 150π; Volume = 25π * 10 = 250π (approx 785.4 if π ~3.14).

A (right circular) cone has base radius r and vertical height h. It also has a slant height ‚Ñì (the length of the straight line from the tip along the surface to a point on the edge of the base). These are related by Pythagoras:

Surface Area and Volume of a Sphere:

A sphere of radius r (like a perfect ball).

Volume: V=43πr3

Conversion of One Solid to Another Shape: This concept involves reshaping an object of a certain volume into a different shape. The key idea is conservation of volume (assuming no material is lost or added).

If you melt or recast a solid into another shape, Volume of original = Volume of new shape (since material amount stays same).

To solve problems: set the volumes equal and solve for the unknown dimension. Example: Suppose you have a metal sphere of radius 3 cm. It‚Äôs melted and recast into a cylinder of height 4 cm. Find the radius of the cylinder.

Another scenario: multiple smaller shapes formed from one big shape or vice versa. For instance, ‚ÄúHow many small cones of given size can be made from a larger cone‚Äôs material?‚Äù ‚Äì use volume ratio to find count.

Surface Area and Volume of Combination of Solids: Sometimes shapes are combined (either attached or one cut out from another).

Volume: If solids are attached (like a capsule shape = cylinder + 2 hemispheres at ends, or a cone on a cylinder like an ice cream cone shape), the total volume is the sum of individual volumes (since volume is just additive for distinct regions).

If one shape is removed from another (like a cylindrical hole drilled through a cube), then volume left = volume(original) - volume(removed part).

For example, a cylinder with a hemisphere on top (dome) ‚Äì total external surface = curved area of cylinder + area of its bottom (if not resting on ground) + curved area of hemisphere + (note: the base of hemisphere and top of cylinder join, so that circle is not exposed externally). So we must exclude that joint circle from surface area calculation.

If asked for surface area of the combination, carefully consider which faces are exposed.

For a shape like a cube with a cylindrical hole, the surface area includes the walls of the hole too because they are now ‚Äúexposed‚Äù inner surface. So you‚Äôd add cylindrical inner surface area and subtract the areas of faces that got holes (the circular openings).

Example (attached solids): A cylinder of radius r and height h with a cone (same base radius r) of height H on top. Volume = πr2h + (1/3)πr2H. Surface area = (curved area of cylinder) + (curved area of cone) + (area of bottom of cylinder). The circular area where cone meets cylinder is not exposed. If bottom sits on a surface and is not exposed, we omit that as well.

Example (removed): A cube of side a with a spherical cavity of radius r bored inside (completely enclosed). Volume remaining = a3 - 4/3 π r3. Surface area exposed = 6a2 (outer faces of cube) plus the inner surface area of the sphere (4πr2) if you consider the inside of cavity as an exposed surface (though if it's completely sealed inside, one might not count it as "surface" unless specifically mentioning inner surface). Usually, combination surface area implies outer surface unless explicitly asking for inner.

7. Data Handling

Data handling encompasses statistics (collecting, organizing, summarizing data) and probability (measuring likelihood of events). It equips us to make sense of numerical information and randomness in daily life.

Collection and Classification of Data: Data can be collected through surveys, experiments, or observation. Once collected, data needs to be organized for clarity.

Raw data: The initial unordered list of observations (e.g., test scores of students in a class). Raw data is often messy or ungrouped.

Classification: We might categorize data (e.g., tally how many falls in certain categories or intervals). For example, if collecting favorite colors of students, we categorize by color; if collecting ages, we might group into ranges (0-10, 10-20, etc.).

Good data handling involves defining what you‚Äôre measuring (ensuring consistency), and possibly designing a table or form to record data systematically.

Frequency Distribution Table: A frequency distribution table lists data values (or intervals of values) against their frequency (how often each occurs). This helps condense raw data.

For discrete data (distinct values, like shoe sizes or number of siblings), you can list each value and count occurrences.

For continuous data or large range, data is often grouped into class intervals (e.g., 0‚Äì10, 10‚Äì20, etc.) and then frequencies counted for each interval.

A typical table might have columns like: Class Interval | Tally | Frequency. Example: Survey of ages of 30 people might produce: 0‚Äì10: 4 people, 11‚Äì20: 5 people, 21‚Äì30: 12 people, 31‚Äì40: 6 people, 41‚Äì50: 3 people.

The table makes it easier to see distribution (most people are in 21‚Äì30 range in this example).

Tally Marks: Tally marks are a quick way to count frequencies by marking strokes (||||) usually in groups of five (the fifth mark crosses the previous four, making it easy to see groups of 5).

When collecting data real-time, one often makes a tally mark for each occurrence in the appropriate category, grouping in fives for easy counting. Example: Counting coin flips results: Heads vs Tails. You could tally: Heads: |||| (4), Tails: |||| | (6). This is quicker to note and then convert to frequency (Heads 4, Tails 6).

In frequency tables, often the tally column is filled first, then tallies are counted to give numeric frequency.

Bar Graph: A bar graph (or bar chart) is a visual representation of categorical data with rectangular bars. Each category‚Äôs bar length (or height) is proportional to its frequency or value.

Bars can be vertical or horizontal. There should be equal spacing between bars (for clarity) and equal width of bars.

Useful for comparing discrete categories (e.g., population of different cities, votes for options, etc.).

Example: If one is showing number of students choosing different stream subjects: Science: 30, Commerce: 20, Arts: 15, a bar graph with those categories on x-axis and count on y-axis will have bars at heights 30, 20, 15 respectively.

Grouped bar graphs can show sub-categories side by side within a category (e.g., male vs female in each class section). Stacked bar charts show totals split into parts. But the basic idea remains comparing magnitudes.

Pictograph: A pictograph represents data using icons or pictures where each icon represents a certain number of units. It‚Äôs a more visually intuitive but less precise way to show data (commonly used in lower grades or infographics).

The key must be given (e.g., ‚ú© = 10 items). Partial icons can represent fractions of that value if needed.

Example: If showing fruit preference among kids with a pictograph: an icon of an apple might represent 5 kids liking apples. If 12 kids like apples, you draw 2 whole apple icons and a partial (to represent 2 out of 5).

Pictographs are simple and fun but not practical for large numbers or precise data.

Pie Diagram (Pie Chart): A pie chart is a circular chart divided into slices, where each slice represents a category‚Äôs proportion of the whole. The whole pie is 100% (or total frequency).

The angle of each sector = Frequency of category}Total frequency}×360°

Good for showing percentage or proportional data (like budget breakdown, market share, etc.) at a glance. However, comparing slices can be tricky if many categories or similar sizes.

Example: If a family's monthly expense is divided into Rent 25%, Food 30%, Education 20%, Savings 15%, Misc 10%, a pie chart will have slices with those percentages of the circle (90¬∞, 108¬∞, 72¬∞, 54¬∞, 36¬∞ respectively for each category).

It‚Äôs important that categories are parts of a whole in a meaningful way (pie chart not appropriate if categories don‚Äôt sum to a meaningful total).

Mean (Average) for Ungrouped Data: The mean is the sum of all observations divided by the number of observations. It‚Äôs a measure of central tendency.

Formula: x¯=∑i=1nxin.

Example: Test scores {70, 80, 95, 60}, mean = (70+80+95+60)/4 = 305/4 = 76.25.

Mean uses all data points, but can be sensitive to outliers (extremely high or low values skew it).

Median for Ungrouped Data: The median is the middle value when data is arranged in ascending (or descending) order. If there are an odd number of observations, median is the center one. If even number, median is the average of the two middle values.

Example: Data {5, 8, 12, 20, 22} (already sorted), median = 12 (3rd of 5). If data {5, 8, 12, 20}, median = (8+12)/2 = 10 (average of 2nd and 3rd in sorted list).

Median is a better measure than mean for skewed distributions or when outliers are present, because it depends on order, not magnitude of extreme values.

Mode for Ungrouped Data: The mode is the value that appears most frequently in the data set. There can be more than one mode (bimodal, multimodal) if multiple values tie for highest frequency, or no mode if all values occur only once (or tie equally in frequency in a trivial way).

Example: Data {4, 1, 4, 2, 7, 4, 2} ‚Äì sorted {1,2,2,4,4,4,7}. The mode is 4 (since 4 occurs 3 times, more than any other number). Another set {1,2,2,3,3} has two modes: 2 and 3 each appear twice.

Mode is the only measure of central tendency that can be used for categorical data (like finding the most popular color among choices).

Mean, Median, Mode for Grouped Data: When data is grouped, exact values are not all known individually, so we use methods to estimate:

Mean: Use the class midpoint (the average of class interval endpoints) as representative of each class, multiply by class frequency to get a "total", sum up and divide by total frequency. Formula: x¯=∑i=1kfixi∑i=1kfi, where xi is midpoint of class i and fi its frequency.

Median: For grouped data, you identify the median class (where the cumulative frequency crosses n/2). Then interpolate within that class using formula:

M=L+(n2-C)×hfm where L = lower boundary of median class, C = cumulative frequency up to class before median class, fm = frequency of median class, h = class width. (This formula finds how far into the median class the median lies, proportional to how many into that class the median falls.)

Mode: Identify the modal class (with highest frequency). The mode can be estimated by formula (for continuous grouped data) known as the mode formula:

Mode=L+(f1-f0)×h2f1-f0-f2 where L = lower boundary of modal class, f1 = frequency of modal class, f0 = freq of class before modal, f2 = freq of class after modal, h = class width. This gives a value within the modal class as an estimate of the "peak". If distribution is not too grouped or classes equal width, sometimes one might just report the modal class or the class mark as the mode.

Grouped formulas are part of higher grade curriculum and needed when precise data isn‚Äôt available, but an estimate is acceptable.

Cumulative Frequency Table: Cumulative frequency (cf) is the running total of frequencies as you move through the classes or values in order. There are two types:

Less-than cumulative frequency: At a value or class boundary X, how many data points are <= X. You accumulate from start up to that point.

Greater-than cumulative frequency: How many are >= a certain value, accumulating from top down. Typically, a cumulative frequency table for grouped data adds another column where each entry is the sum of that class frequency and all previous class frequencies. Example: Suppose class intervals 0‚Äì10 (f=5), 10‚Äì20 (f=8), 20‚Äì30 (f=7), 30‚Äì40 (f=10). Less-than cumulative freq:

up to 10: 5

up to 20: 5+8 = 13

up to 30: 13+7 = 20

up to 40: 20+10 = 30 (which should equal total n). These help in finding median or percentiles (e.g., median is at n/2 = 15th data, which falls in 20‚Äì30 class here). Plotting cumulative frequency against upper class boundary yields an ogive.

Ogive Curves: An ogive is a graph of the cumulative frequency distribution. There are typically two:

Less-than ogive: Plot points (upper class boundary, cumulative frequency) and join them with a smooth curve or straight line segments. It starts at the lowest boundary with cf = 0 and climbs to total n at highest boundary.

Greater-than ogive: Plot (lower class boundary, cf downward) which starts at total n and decreases to 0 at the top end. If both are drawn, their intersection gives median (but often just one type is drawn).

Ogives are useful to estimate median, quartiles, etc., by reading values corresponding to 25%, 50%, 75% of total frequency on the y-axis and dropping to x-axis.

The shape is typically an S-shaped curve for less-than ogive (starting slow, steep in middle, leveling at end if distribution is reasonable). It‚Äôs basically the cumulative distribution graph.

Concept of Probability: Probability is a measure of how likely an event is to occur, on a scale from 0 (impossible) to 1 (certain). For equally likely outcomes, the probability of an event = (number of favorable outcomes) / (total number of outcomes).

Experiment, Sample Space, Event:

An experiment is a process that yields random results (e.g., tossing a coin, rolling a die).

The sample space (S) is the set of all possible outcomes. E.g., for a die S={1,2,3,4,5,6}.

An event is a subset of the sample space ‚Äì one or more outcomes that satisfy a condition (e.g., event A = ‚Äúroll an even number‚Äù = {2,4,6}).

Probability of event A: P(A) = |A|/|S| if outcomes are equally likely. Example: probability of even on a die = 3/6 = 1/2.

Probabilities can also be thought of in terms of long-run frequency: if you repeat the experiment many times, the fraction of times A occurs should approach P(A).

Probability values: always between 0 and 1 inclusive. Sum of probabilities of all elementary outcomes = 1.

Simple Problems on Single Events: Day-to-day probability often involves one event or a single trial scenario: Examples:

Tossing a fair coin: P(Heads) = 1/2, P(Tails) = 1/2.

Drawing a card from a well-shuffled standard deck (52 cards): P(drawing an Ace) = 4/52 = 1/13; P(drawing a Heart) = 13/52 = 1/4.

Rolling a fair die: P(prime number) = favorable outcomes {2,3,5} = 3/6 = 1/2; P(>4) = outcomes {5,6} = 2/6 = 1/3.

Picking a random person and finding probability they have a birthday in January = 31/365 (ignoring leap). These are single-event probabilities. We assume each outcome is equally likely in such simple scenarios.

Complementary Events: The complement of event A, often denoted A‚Ä≤ or A^c, is the event that A does not happen (i.e., an outcome is in the sample space S but not in A).

Property: P(A) + P(A‚Ä≤) = 1. Because either A happens or it doesn‚Äôt (assuming there's no other possibility). So, P(A‚Ä≤) = 1 - P(A).

Using complement often simplifies calculations. For example, if we want the probability of at least one success in several trials, it‚Äôs easier to do 1 - P(no successes). Or probability of ‚Äúat least one 6 in two die rolls‚Äù = 1 - P(no 6 in either roll) = 1 - (5/6 * 5/6).

Example: If the chance of rain today is 0.3 (30%), then the chance of not raining (the complement) is 1 - 0.3 = 0.7 (70%).

For a random person, if event A = ‚Äúborn on a weekend‚Äù, P(A) say = 2/7 (approx .2857), then complement ‚Äúborn on weekday‚Äù probability = 5/7 (~.7143).

Complementary events are mutually exclusive and exhaustive (they cover all outcomes between them, and can‚Äôt occur together).

8. Trigonometry

Trigonometry is the study of the relationships between angles and sides in triangles, especially right-angled triangles. It introduces trigonometric ratios that are fundamental in geometry, physics, engineering, and many applied fields. Key concepts include:

Introduction to Trigonometry (Trigonometric Ratios): Consider a right triangle (one angle = 90¬∞). Focus on one of the acute angles, call it θ. Relative to θ:

The side opposite θ is the side across the triangle from θ.

The side adjacent to θ is the side next to θ (forming the angle θ, other than the hypotenuse).

The hypotenuse is the side opposite the right angle (the longest side of the right triangle). The primary trigonometric ratios are defined as:

sin θ (sine) = (length of Opposite side) / (length of Hypotenuse).

cos θ (cosine) = (length of Adjacent side) / (length of Hypotenuse).

tan θ (tangent) = (length of Opposite side) / (length of Adjacent side). These definitions come from similar right triangles: for a given angle θ, the ratio of specific sides is constant regardless of triangle size.

Additionally, the reciprocals of these are defined:

cosecant (csc or cosec θ) = 1/sinθ = Hypotenuse/Opposite.

secant (sec θ) = 1/cosθ = Hypotenuse/Adjacent.

cotangent (cot θ) = 1/tanθ = Adjacent/Opposite.

Example: In a right triangle, if angle θ is such that opposite side = 3, adjacent = 4, hypotenuse = 5 (a 3-4-5 triangle), then sinθ = 3/5 = 0.6, cosθ = 4/5 = 0.8, tanθ = 3/4 = 0.75.

Relationship Between Trigonometric Ratios: Using the above definitions, certain identities and relationships are evident:

tan θ = sin θ / cos θ. (Because sinθ = opp/hyp and cosθ = adj/hyp, dividing gives opp/adj.)

cot θ = cos θ / sin θ (similarly, or cot = 1/tan).

sec θ = 1/cos θ, cosec θ = 1/sin θ.

Primary Pythagorean Identity: sin2θ+cos2θ=1 This comes from the Pythagorean theorem: (opp/hyp)2 + (adj/hyp)2 = (opp2+adj2)/hyp2 = hyp2/hyp2 = 1. From this, we can derive:

sec2θ=1+tan2θ and cosec2θ=1+cot2θ

Other Pythagorean identities: Divide the primary identity by cos2θ (assuming cosθ≠0) to get 1+tan2θ=sec2θ Divide by sin2θ to get 1+cot2θ=cosec2θ.

These fundamental relationships allow transforming expressions and solving trigonometric equations.

Values of Trigonometric Ratios (Specific Angles): Certain angles (0¬∞, 30¬∞, 45¬∞, 60¬∞, 90¬∞) have well-known sine, cosine, and tangent values worth memorizing for quick reference:

0¬∞: sin 0¬∞ = 0, cos 0¬∞ = 1, tan 0¬∞ = 0.

30¬∞: sin 30¬∞ = 1/2, cos 30¬∞ = √3/2 (~0.8660), tan 30¬∞ = 1/√3 (~0.5774).

45¬∞: sin 45¬∞ = √2/2 (~0.7071), cos 45¬∞ = √2/2 (~0.7071), tan 45¬∞ = 1.

60¬∞: sin 60¬∞ = √3/2 (~0.8660), cos 60¬∞ = 1/2, tan 60¬∞ = √3 (~1.732).

90¬∞: sin 90¬∞ = 1, cos 90¬∞ = 0, tan 90¬∞ is undefined (since cos 90¬∞ = 0, tan = sin/cos would involve division by zero; in a right triangle, the angle approaches 90¬∞, opposite side ~ hypotenuse, adjacent side -> 0, so tan gets infinitely large).

These can be recalled by various mnemonics or constructed via special triangles: e.g., 30-60-90 triangle (sides in ratio 1:√3:2) for 30¬∞ and 60¬∞, and isosceles right triangle (1:1:√2) for 45¬∞.

Also, cos is the complementary of sin in these values (cos 30¬∞ = sin 60¬∞, cos 45¬∞ = sin 45¬∞, etc.), which leads into the complementary angles concept next.

Trigonometric Identities: We already listed the Pythagorean identities. There are also angle sum and difference identities (for advanced usage, like sin(A±B) formulas), double angle formulas, etc., but within basic curriculum, the most important identities to remember are:

sin2θ + cos2θ = 1.

1 + tan2θ = sec2θ.

1 + cot2θ = cosec2θ.

sin(90¬∞ - θ) = cos θ, cos(90¬∞ - θ) = sin θ (complementary angles, see below).

tan(90¬∞ - θ) = cot θ, cot(90¬∞ - θ) = tan θ.

Additionally, sin(-θ) = -sin θ, cos(-θ) = cos θ (even/odd function property; negative angle flips sine‚Äôs sign but not cosine‚Äôs). Tan(-θ) = -tan θ. These reflect symmetry of trig functions: sine and tangent are odd, cosine is even.

Basic identities also include sin2θ=2sinθcosθ, cos2θ=cos2θ-sin2θ (though these might be slightly beyond intro level, they come in handy in higher classes).

Trigonometric Ratios of Complementary Angles: Complementary angles sum to 90¬∞. In a right triangle, aside from the 90¬∞, the other two angles are complementary to each other. Because of how sine and cosine are defined relative to different acute angles in the same right triangle, we get relationships:

If angles A and B are such that A + B = 90¬∞, then B = 90¬∞ - A.

sin A = cos(90¬∞ - A) = cos B.

cos A = sin(90¬∞ - A) = sin B.

tan A = cot(90¬∞ - A) = cot B.

cot A = tan(90¬∞ - A) = tan B.

sec A = cosec(90¬∞ - A), and cosec A = sec(90¬∞ - A). Example: sin 30¬∞ = cos 60¬∞, since 60¬∞ is comp of 30¬∞. Indeed sin30 = 0.5, cos60 = 0.5. Or tan 10¬∞ = cot 80¬∞. This makes sense because in one right triangle, one acute angle‚Äôs ‚Äúopposite‚Äù is the other‚Äôs ‚Äúadjacent‚Äù and vice versa. So one‚Äôs sine is the other‚Äôs cosine, etc.

Applications of Trigonometry: Trigonometry allows us to solve problems involving indirect measurements ‚Äì typically lengths that are hard to measure directly can be computed using angles and a reference length. A classic application is measuring heights and distances using angles of elevation or depression:

Angle of Elevation: The angle your line of sight makes with the horizontal when you look up at an object. For instance, if you stand some distance from a tall building and look up to the top, the angle between a horizontal line from your eyes and the line of sight to the top is the angle of elevation.

Angle of Depression: The angle of your line of sight below the horizontal when looking down at an object. E.g., from the top of a lighthouse looking at a boat at sea, the angle between horizontal and the line of sight down to the boat is angle of depression. (Note: Angle of depression from one point equals angle of elevation from the other point if lines are parallel, like horizontal at observer and horizontal at object‚Äôs level, by alternate interior angles.)

Solving height-distance problems: Usually involves right triangles: one angle is known (elevation/depression), one side (often horizontal distance or vertical height) is known or measurable, and you need to find the other side.

If you know the distance from the object and the angle of elevation, you can find height using: tan(θ) = opposite/adjacent, so opposite (height difference) = adjacent * tan(θ). Example: You are 50 m from a tower, angle of elevation to top is 30¬∞ ‚Üí height of tower above your eye level = 50 * tan30¬∞ ‚âà 50 * 0.577 = 28.9 m. Add your eye height if needed for total height.

If you know height and need distance, use tan again: distance = height / tan(θ).

Alternatively, sometimes use sin or cos if appropriate: e.g., a ladder leaning to a wall makes angle with ground, one might find how high it reaches (vertical) by using sin or cos.

Angle of Depression example: From a plane at 1000 m altitude, pilot sees a target on ground at angle of depression 30¬∞. That means the line from plane to target is 30¬∞ below horizontal, which is essentially a 30¬∞ angle in a right triangle with vertical 1000 m (opposite) and slant distance as hypotenuse or horizontal distance as adjacent. Typically, horizontal distance = 1000 * cot30¬∞ = 1000 * (√3) ‚âà 1732 m. Or the slant distance (line of sight) = 1000 / sin30¬∞ = 2000 m.

In all such problems, it‚Äôs crucial to draw a diagram, mark known lengths and angles, choose the right trig ratio (sin for opposite/hypotenuse, cos for adjacent/hypotenuse, tan for opposite/adjacent) and solve.

Trigonometry is also used in navigation, architecture (roof slopes etc.), engineering (forces components), and so forth, but at this level, it‚Äôs mostly these right-triangle angle problems.
